# Trabalho_2_IA
Este repositório contém o desenvolvimento e os experimentos práticos do Trabalho 2 da disciplina de Inteligência Artificial (período 2026.1) da Universidade Federal do Ceará - Campus de Quixadá. O objetivo do projeto é aplicar, analisar e comparar diferentes algoritmos de busca (heurísticas e locais) em um ambiente controlado de grade
